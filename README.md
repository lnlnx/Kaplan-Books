# KTP book list sample project

## Running the application

```javascript
# first install npm packages
npm install 
npm start
```

## Folder structure

- src
  - components
    - Header
    - NewBook
    - MainSection
      - Search
      - Booklist
        - Book
    - UI
      - Backdrop
      - Button
      - Modal
  - styles
    - _shared.scss (for globall css variables)
  - store
    - actions
    - reducers
    - sagas


## Libraries used:
- React
- Redux
- Redux Saga
- SCSS & BEM
- Jest for testing
- Enzyme (for testing React components)
